1) Overview

Before we start, lets take a look on the folder structure of the theme. All theme files are placed in the HTML folder. The main HTML folder consists of a folder named resources 4 more folders for diferent types of theme assets:


  1. css - containing all CSS files for the theme.
  2. fonts - containing all fonts files for the theme.
  3. Images - containing all images in the background slider..
  4. js -containing all scripts for the jquery effects.

2) Installation

This is an HTML theme and doesn't require instalation in the traditional meaning of the word. You only need to place the content (HTML files and the three folders) in your prefered web directory

Keep in mind that the folder structure and file/folder names are important for the correct theme work



3) HTML Structure

The main HTML structure of the template consists of 5 main containers containing all page ellements:


  1. Header
  2. Countdown
  3. About section
  4. Contact section
  5. Footer

All 5 containers are shown with comments and well managed, which makes it easy to customize








4) CSS files and structure

The theme comes with several CSS (stylesheet) files grouped in the stylesheet folder:

  1. flexslider.css - to allow CSS3 animations on the caption every time a slide is changed.
  2. font-awesome.css - font-awesome css file
  3. gumby.css - a flexible, responsive CSS framework
  4. style.css - the main CSS file containg the layout styling of the pages
  5. superslider.css - a fullscreen slider for jQuery
  6. zocial.css - css file for buttons & icons styles

5) JavaScript

This template is heavily loaded with both custom JS functions and free jQuery plugins. All JS files are placed in a separate folder called "js". The list of files is as follows:

  1. custom.js
  2. jquery.js
  3. jquery.countdown.min.js (inside js/new folder)
  4. jquery.flexslider-min.js (inside js/new folder)
  5. jquery.superslides.min (inside js/new folder)
  6. plugins.js
  7. spin.js

some of the jQuery plugins are heavily customized and chages in their files are not recommended. However if you wish to modifie the time, just follow the steps below :

  1. Open the index.html file"
  2. The script is "/* Count down */"
  3. And replace "2015/03/06" with the required date

    4. And if you wish to change your location, just follow the steps below :

    1. Open the plugins.js file"
    2. The first "7 lines"
    3. And change "Latitude&Longitude " with the required location
